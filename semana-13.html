SEMANA Nº13

Optimización del prototipo

Tras las evaluaciones pasadas, se procedió a re-diseñar los módulos para eliminar las partes innecesarias, pulir algunos bordes y arreglar el tamaño de los agujeros.

Soporte principal

Para el soporte principal las modificaciones fueron de un grado menor. Se aumentó la densidad de impresión y se amplió el agujero para que pueda encajar el remache. Además se aplicó silicona para que sirva como tope de giro, dándole firmeza al movimiento.

 

 

Brazalete

Debido a que se ha optado por solo uno de los soportes, ya no será necesario contar con el sistema de agarre diseñado para el soporte descartado, por ende se removió esa parte y, además, se amplió el diámetro del agujero por donde se sujeta el soporte 1, para en lugar de colocar un tornillo se unan a través de un remache. También se eliminó el sistema pensado para unir ambas partes del brazalete, y en su lugar se utilizará velcro por ser mas práctco.

Guante

Se corrigieron el largo del guante, asimismo se corrigió la posición del velcro para que se adhiera al brazalete de forma más sencilla.

Se colocó foam entre capas de tela, justo por debajo de donde encaja el brazalete, de esta forma el brazalete se soluciona el ajuste en muñecas muy delgadas y se mejora la comodidad en caso quede ajustado.
